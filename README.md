# 김민재

추천 시스템과 데이터 기반 서비스 구현에 관심이 있습니다.  
사용자 행동 데이터를 분석해 문제를 정의하고, 추천/랭킹/챗봇 흐름을 서비스 형태로 연결해왔습니다.  
모델링뿐 아니라 데이터 수집, API, 웹, 배포까지 이어지는 구현 경험을 쌓고 있습니다.

## 대표 프로젝트

### fcoach

- **무엇을 만들었나**: FC Online 경기 데이터를 분석해 플레이 문제를 진단하고, 액션 제안과 효과 검증까지 연결한 데이터 기반 코칭 서비스
- **핵심 기술**: `Next.js`, `FastAPI`, `SQLite`, `Nexon Open API`, `FC Online DataCenter`
- **성과 / 특징**: 데이터 수집 -> KPI 계산 -> 분석 API -> 웹 화면 -> Vercel 배포까지 직접 구현
- **링크**: [GitHub](https://github.com/minzai0116/fcoach) · [서비스](https://fcoach.fun)

### RMSE: Ronaldo Market Search Engine

- **무엇을 만들었나**: 자연어 요청을 검색과 추천이 가능한 형태로 해석해 전자제품을 추천하는 대화형 추천 시스템
- **핵심 기술**: `Retrieval`, `Reranking`, `LLM`, `CatBoost`, `PyTorch`, `FastAPI`, `PostgreSQL`, `Redis`
- **성과 / 특징**: Amazon Reviews'23 기반 대규모 리뷰 데이터에서 Retrieval + Reranking + LLM 구조의 추천 파이프라인 설계
- **링크**: [GitHub](https://github.com/minzai0116/pro-recsys-finalproject-recsys-06) · [발표 영상](https://youtu.be/SAFLFog-qMg)

### Sound_of_Heart

- **무엇을 만들었나**: 정서적 어려움을 겪는 사용자를 위한 상담형 챗봇과 커뮤니티 기반 멘탈케어 서비스
- **핵심 기술**: `LLM Prompting`, `Chatbot Flow`, `CBT`, `ACT`, `DBT`, `Community`
- **성과 / 특징**: 챗봇 구조, 프롬프트 구성, 상담 이론 반영, 커뮤니티 기능 설계에 기여
- **링크**: [GitHub](https://github.com/minzai0116/Sound_of_Heart)

## 관심 분야

- 추천 시스템과 랭킹 모델
- 사용자 행동 데이터 분석
- 데이터 기반 서비스 설계
- LLM 기반 대화형 추천과 상담형 챗봇
- 모델링 결과를 API와 웹 서비스로 연결하는 구현
